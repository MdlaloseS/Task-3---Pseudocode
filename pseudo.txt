Answer #1. Largest number until zero

request a positive number from the user
set largest number to 0

while the number is not equal to 0
    if the number is greater than the largest number
        set largest number to the number
    request another positive number from the user

print the largest number entered


Answer #2. Store name and print greeting

request the user's name
store the name in a variable called first_name
print first_name and "Hello, World"


Answer #3. Arithmetic average of integers

set sum to 0
set count to 0

request an integer from the user

while the integer is not equal to 0
    add the integer to sum
    increase count by 1
    request another integer from the user

if count is greater than 0
    calculate average as sum divided by count
    print the average
else
    print "no numbers were entered"


Answer #4. Grocery list in alphabetical order

create an empty list called grocery_list

request a grocery item from the user

while the item is not equal to "done"
    add the item to grocery_list
    request another grocery item from the user

sort grocery_list in alphabetical order

for each item in grocery_list
    print the item
